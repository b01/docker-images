## Contributing
If you want to share your images, then just create a folder
in the root and name it after the OS. In that folder, it is recommended you
make additional sub folders for each image you want to make.


## How to Use

./remove-all.(sh|ps1)
./build-all.(sh|ps1)
./push-all.(sh|ps1)

For example:
```bash
./build-all.sh
```
